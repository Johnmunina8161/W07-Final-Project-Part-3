# Event Management API - v2

This version includes:
- Full CRUD with validation and error handling
- OAuth (Google) ready using passport-google-oauth20
- Jest + Supertest unit tests for GET routes
- Swagger docs at /api-docs

Quick start:
1. Copy .env.example to .env and fill credentials.
2. npm install
3. npm run dev
4. Visit http://localhost:3000/api-docs
